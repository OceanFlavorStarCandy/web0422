ABOUT ME

I like coding

저는 대학교를 졸업할 때 게임 UI 디자이너를 목표로 하고 있었습니다. 본격적으로 취업 준비를 하기 위해, 국민취업지원제도를 통해 게임 UI 관련 수업을 찾아보았으나 관련된 강의가 별로 없었습니다. 그렇게 들을 수업을 찾던 중 우연한 기회로 웹디자인 수업을 듣게 되었습니다. 평소에 좋아하던 게임의 업데이트 소개 페이지 UI를 보면서 웹디자인에도 관심이 있었기에, 웹디자인과 퍼블리싱 쪽으로 진로를 변경하려고 마음 먹었습니다.
6개월 동안 포토샵과 figma를 이용해 웹페이지 와이어프레임과 디자인 시안을 제작하고, html, css와 자바스크립트로 구현하는 훈련을 받으며 실무에 필요한 기술을 탄탄히 다졌습니다. 이 과정을 통해 웹디자인은 저에게 단순한 진로가 아닌, 제가 꿈꿔왔던 시각적 창의성을 발휘할 수 있는 최적의 분야라는 확신을 얻었습니다.

Eunji Kim

웹디자인 수업 때 팀 프로젝트를 2번 했었습니다. 첫번째 프로젝트는 호텔 신라 페이지의 리디자인으로 헤더와 푸터 제작, 메인 페이지(대문), 회사 소개 페이지, 멤버십 혜택 안내 페이지를 제작하였습니다. 두번째 프로젝트는 설빙 브랜드의 웹 페이지 리디자인으로 반응형 메뉴 페이지, FAQ 게시판 페이지 제작을 담당했습니다. 이러한 프로젝트를 통해, 저는 다양한 종류의 페이지 디자인과 퍼블리싱 경험을 쌓았고, 클라이언트의 요구에 맞춘 최적화된 디자인과 기능을 구현할 수 있습니다.
저는 작업을 할 때 우선 클라이언트의 요청 사항을 확인하고, 그 후 벤치마킹할 웹 사이트나 이미지 자료를 서치합니다. 그 후 와이어프레임으로 웹 페이지에 들어갈 콘텐츠 위치 등을 구체적으로 설정합니다. 와이어프레임 설계 후에는 그것을 토대로 디자인 시안을 피그마, 포토샵을 이용해 만듭니다. 이 과정에서 클라이언트의 피드백을 반영하며 수정 및 보완을 하여 시안을 완성합니다.
시안이 완성되면 이를 바탕으로 HTML, CSS를 활용해 페이지를 코딩하고, 동적인 부분이나 기능 구현이 필요한 곳에는 자바스크립트를 이용해 웹 페이지를 구현합니다. 특히, 설빙 프로젝트에서 반응형 웹 디자인을 통해 모바일과 데스크톱 환경 모두에서 최적화된 사용자 경험을 제공할 수 있도록 세심한 설계를 했고, 이를 통해 사용자 만족도를 크게 높일 수 있었습니다.
또한, 캐릭터 제품 촬영 및 이미지 보정에도 경험이 있습니다. 지인의 요청으로 폰 케이스 커스텀이나 의상에 필요한 이미지 가공 및 제작을 해본 경험이 있어, 이미지 보정 및 시각적 퀄리티 향상에도 자신감을 가지게 되었습니다.
저는 이러한 경험을 바탕으로, 클라이언트의 요구를 충실히 반영하면서도 차별화된 디자인을 구현할 수 있는 웹디자이너로 성장하고 있습니다. 실무에서도 이를 통해 고객의 요구를 만족시키는 웹 사이트 제작에 기여하고 싶습니다.
